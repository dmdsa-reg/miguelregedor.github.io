--- layout: post title: Git clone from remote repositorium with ssh and non default port --- just a reminder, suppose your ssh port is 3022: (sometimes I swap the way to specify the port in those two commands) 
git clone ssh://user@domain.com:3022/~/Projects/my_project

ssh user@domain.com -p 3022
(when connecting through ssh the non default port goes as param, the default port is 22)
